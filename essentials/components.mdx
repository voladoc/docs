# Mintlify 组件库

Mintlify 提供了丰富的内置组件，可以帮助你创建美观、交互性强的文档。本指南将介绍最常用的组件及其用法。

## 布局组件

### 卡片 (Card)

卡片组件是展示内容最常用的方式之一，支持标题、图标、链接和内容。

<Card title="这是一个卡片标题" icon="star">
  这是卡片内容，可以包含任何 Markdown 或组件。
</Card>

**使用方法：**

```jsx
<Card title="这是一个卡片标题" icon="star">
  这是卡片内容，可以包含任何 Markdown 或组件。
</Card>
```

### 卡片组 (CardGroup)

卡片组可以将多个卡片组织在一起，支持设置列数。

<CardGroup cols={2}>
  <Card title="第一张卡片" icon="bolt">
    第一张卡片的内容
  </Card>
  <Card title="第二张卡片" icon="book">
    第二张卡片的内容
  </Card>
  <Card title="第三张卡片" icon="globe">
    第三张卡片的内容
  </Card>
  <Card title="第四张卡片" icon="code">
    第四张卡片的内容
  </Card>
</CardGroup>

**使用方法：**

```jsx
<CardGroup cols={2}>
  <Card title="第一张卡片" icon="bolt">
    第一张卡片的内容
  </Card>
  <Card title="第二张卡片" icon="book">
    第二张卡片的内容
  </Card>
</CardGroup>
```

  <Column>

    这是第二列的内容，可以包含任何 Markdown 或组件。

**使用方法：**

```jsx
<Columns>

    ## 第一列
    这是第一列的内容

  <Column>
    ## 第二列
    这是第二列的内容

## 交互组件

### 标签页 (Tabs)

标签页可以在同一区域显示不同的内容，用户可以通过点击标签切换。

<Tabs>
  <Tab title="HTML">
    ```html
    <div class="example">
      <h1>Hello World</h1>
    </div>
    ```
  </Tab>
  <Tab title="CSS">
    ```css
    /* CSS 代码 */
    .example {
      color: blue;
    }
    ```
  </Tab>
  <Tab title="JavaScript">
    ```javascript
    function greet() {
      console.log('Hello World!');
    }
    ```
  </Tab>
</Tabs>

**使用方法：**

```jsx
<Tabs>
  <Tab title="HTML">
    ```html
    <div class="example">
      <h1>Hello World</h1>
    </div>
    ```
  </Tab>
  <Tab title="CSS">
    // CSS 代码
  </Tab>
</Tabs>
```

## 常见问题

<AccordionGroup>
  <Accordion title="如何使用自定义图标？">
    Mintlify 支持 FontAwesome 图标库。在组件的 `icon` 属性中使用图标名称即可。
    
    例如：`<Card icon="star" />`
  </Accordion>
  
  <Accordion title="可以在组件中嵌套其他组件吗？">
    是的，大多数 Mintlify 组件都支持嵌套。例如，你可以在 Card 组件中嵌套 Tabs 组件，或者在 Step 组件中嵌套 CodeGroup 组件。
  </Accordion>
  
  <Accordion title="如何自定义组件样式？">
    Mintlify 组件支持在 `docs.json` 中配置主题样式，或者通过 CSS 类进行自定义。详细信息请参考 Mintlify 官方文档中的主题设置部分。
  </Accordion>
</AccordionGroup>

## 高级用法

### 组件属性

大多数组件都支持以下属性：

- **className**: 自定义 CSS 类
- **style**: 内联样式

例如：

```jsx
<Card title="自定义样式" className="custom-card" style={{ border: '2px solid blue' }}>
  这是一个带有自定义样式的卡片。
</Card>
```

### 组件组合

通过组合不同的组件，可以创建复杂且交互性强的文档：

<Steps>
  <Step title="选择你需要的组件">
    <CardGroup cols={2}>
      <Card title="布局组件" icon="columns">
        用于组织页面结构
      </Card>
      <Card title="交互组件" icon="hand-pointer">
        增加页面交互性
      </Card>
    </CardGroup>
  </Step>
  <Step title="组合使用">
    <Tabs>
      <Tab title="简单组合">
        可以将卡片放入列中
        
        ```jsx
        <Columns>
          <Column>
            <Card title="左侧卡片">内容</Card>
          </Column>
          <Column>
            <Card title="右侧卡片">内容</Card>
          </Column>
        </Columns>
        ```
      </Tab>
      <Tab title="复杂组合">
        可以在步骤中使用标签页和代码组
        
        ```jsx
        <Steps>
          <Step title="第一步">
            <Tabs>
              <Tab title="方式一">内容</Tab>
              <Tab title="方式二">内容</Tab>
            </Tabs>
          </Step>
        </Steps>
        ```
      </Tab>
    </Tabs>
  </Step>
</Steps>
